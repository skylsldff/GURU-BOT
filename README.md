# sAura MD WhatsApp Bot

Ein WhatsApp-Gruppenhelfer, der auf private Befehle und Gruppenmoderation spezialisiert ist. Der Bot hilft mit Gruppenbeitritt, Begrüßungen, XP, AFK-Status, Verwaltungsbefehlen und mehr.

## Was der Bot kann
- `/join <chat.whatsapp.com/xxxx>`: Der Bot tritt der Gruppe privat über einen Gruppenlink bei.
- `/help` oder `/-help`: Zeigt eine ausführliche Hilfe mit wichtigen Befehlen.
- `/menu`: Zeigt das Menü mit allen Befehlen.
- `/ping`: Liefert die aktuelle Latenz.
- `/bot`: Erklärt, wer der Bot ist und wie er entstanden ist.
- `/myinfo`: Zeigt dein Profil, XP, Level und Rang.
- `/ranklist`: Zeigt die verfügbaren Ränge.
- `/profile @user`: Sendet ein Profilbild eines markierten Nutzers.
- Gruppenadministration: Kick, Add Picture, Begrüßungen, XP, AFK, Leave usw.

## Projektbeschreibung
sAura MD ist ein WhatsApp-Bot für Gruppe und privaten Chat. Er wurde als Gruppenhelfer entworfen, um Admins zu unterstützen, Regeln durchzusetzen, neue Mitglieder willkommen zu heißen und den Gruppenalltag zu vereinfachen.

Der Bot wurde nach deiner Vorgabe beschrieben als Teil des Projekts Alpha sAura MD. Er ist darauf ausgelegt, größere Gruppen zu unterstützen und den Betrieb zu automatisieren.

## Installation

1. Repo forken oder klonen:
   - Forke das Projekt auf GitHub, falls du eigene Anpassungen machen willst.
   - Klone dann dein Fork oder das Original:
     `git clone https://github.com/<dein-benutzername>/GURU-BOT.git`

2. Wechsle in das Projektverzeichnis:
   `cd GURU-BOT`

3. Installiere die Abhängigkeiten:
   `npm install`

4. Erstelle falls nötig eine Umgebungsdatei basierend auf `sample.env`.

## Starten

1. Starte den Bot:
   `npm start`

2. Scanne den QR-Code mit WhatsApp Web / WhatsApp App, falls er im Terminal angezeigt wird.

3. Schreibe dem Bot privat einen Gruppenlink mit:
   `/join <chat.whatsapp.com/xxxx>`

4. Der Bot tritt der Gruppe bei und sendet eine Begrüßung.

## Forken und eigene Anpassungen

- Forke das Repository, indem du auf GitHub auf `Fork` klickst.
- Klone dann deinen Fork und arbeite lokal weiter.
- Push deine Änderungen zu deinem Fork:
  `git push origin main`
- Wenn du möchtest, kannst du später einen Pull Request zum Original-Repository erstellen.

## Hinweise

- Der Bot akzeptiert nur Gruppenbeitritte über den privaten `/join`-Befehl.
- Direkte Einladungen werden in der Regel abgelehnt oder führen dazu, dass der Bot die Gruppe verlässt.
- Achte darauf, dass der Bot Admin-Rechte hat, wenn er Gruppenadministrationsbefehle ausführen soll.
